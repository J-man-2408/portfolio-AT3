# Jasmine Sullivan - C4 Programming Web Tech Portfolio

A personal portfolio site built with HTML, Tailwind CSS and vanilla JavaScript as part of the Web Dev unit for the Cert IV IT (Programming) assessment

## About this website project

### Pages
- **Home Page** will include a hero image, brief introduction and cards linking to appropriate pages 
- **About Page** will include a overview section, a hobbies & skills section and an aspirations section
- **Contact Page** will include a form to contact me via email 
- **Project Page** will include two pictures of previously made websites and four cards linking to appropriate pages
- **Resume Page** will include my education, my skills, two card linking to projects and a document resume that is downloadable
- **Sample Project Page** will include my API form from Phase 2, an overview of the sample project and an evidence section (a pciture of the website project)
